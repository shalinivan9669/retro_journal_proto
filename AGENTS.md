## Important Gameplay State

The project currently has:

- first-person WASD movement and mouse look;
- center dot interaction cursor, not a crosshair;
- an interactable cube in the main room;
- a 13 second cube memory cutscene started with `E`;
- movement, mouse look, and interaction lock during that cutscene;
- a memory painting that appears on the yurt wall after the cutscene;
- a large invisible cube interaction area to make aiming at the cube easier;
- a large door that transitions to `InfiniteRoad.tscn`;
- an infinite road scene with VHS return behavior.

## Do Not Break

Keep these working unless the user specifically asks to replace them:

- `Main.tscn` loads without errors.
- `Player` can move with WASD and look with mouse.
- `E` interacts through the center dot.
- Cube starts the memory cutscene.
- Door still changes to `res://scenes/levels/InfiniteRoad.tscn`.
- `InfiniteRoad.tscn` still runs.
- VHS / fullscreen overlay work should remain fullscreen.

## Working Style For Future Codex Agents

Before editing, read this file and then inspect only the specific files needed for the task.

Prefer small targeted edits. Use `apply_patch` for manual edits. Keep existing scenes and scripts intact unless the user asks for a rebuild.

When a user asks to make interaction easier, prefer enlarging invisible `Area3D` interaction volumes before enlarging visible meshes.

When a user complains that an image is stretched or cropped, first check `TextureRect.stretch_mode`, material UV scale, mesh proportions, and whether the image has been imported.

When changing gameplay, verify `Main.tscn` and `InfiniteRoad.tscn` with the headless Godot commands above.
Also verify `res://scenes/levels/UndergroundSteppe.tscn` when touching the floor hatch or underground controller.
