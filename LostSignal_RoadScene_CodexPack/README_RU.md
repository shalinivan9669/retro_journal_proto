# Lost Signal — Road Scene Codex Pack

Единый набор для сборки первой сцены ночной поездки к Балхашу в Godot 4.7.

## Содержимое

- `CODEX_PROMPT_RU.md` — готовое задание для Codex.
- `references/` — четыре созданных изображения-референса.
- `materials/road/` — PBR старого асфальта и повреждений.
- `materials/steppe/` — PBR сухой земли и обочины.
- `backdrops/runtime_8k/` — многослойный горизонт, горы, озеро и haze.
- `sky/` — ночное HDRI 4K.
- `docs/` — инструкция по сцене, промпты генерации и лицензии источников.

Главный принцип: изображение дороги служит референсом, а сама дорога строится из повторно используемых 3D-секций с бесшовными PBR-текстурами.
