# Poly Haven sources — CC0

Все перечисленные внешние ассеты опубликованы Poly Haven под CC0.

- Dark Rock: https://polyhaven.com/a/dark_rock
- Plastered Stone Wall: https://polyhaven.com/a/plastered_stone_wall
- Rogland Clear Night: https://polyhaven.com/a/rogland_clear_night

Оригинальные прямые URL и воспроизводимый загрузчик находятся в `tools/download_polyhaven_assets.sh`.

Runtime WebP normal maps являются оптимизированными производными от оригинальных OpenGL normal PNG. Скрипт загрузки позволяет заново получить исходные PNG без потерь.
