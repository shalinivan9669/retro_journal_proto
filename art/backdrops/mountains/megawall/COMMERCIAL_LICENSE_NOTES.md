# Commercial license notes

Safe sources in this package plan:

- Poly Haven: downloadable assets are CC0.
- ambientCG: downloadable assets are CC0.
- OpenHDRI: CC0 HDRIs.
- OpenTopography/Copernicus DEM: free public dataset, but check dataset terms and cite/source in internal documentation.
- NASA/USGS data: generally permissive/public domain, but check each dataset and avoid NASA endorsement/logos.

Do not use as game textures:

- Google Maps / Google Earth tiles;
- Yandex Maps tiles;
- Bing Maps tiles;
- random Pinterest images;
- random ArtStation images;
- CC-NC or CC-ND photos;
- CC-BY-SA photos unless you understand share-alike implications.

Poly Haven API note:

The assets are CC0, but automated API access has separate terms. For commercial production, use manual downloads from asset pages, official tools/add-ons where appropriate, or get API permission if you are building an automated bulk-downloader.
