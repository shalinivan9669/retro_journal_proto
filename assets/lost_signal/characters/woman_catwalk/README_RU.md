# Девушка с модельной походкой — пакет для Godot 4

В проект перенесена основная версия для близкого NPC:

- `woman_catwalk_pbr_animated.glb` — основная версия с PBR-текстурами;

Модель получила новый 23-костный риг. Основная пластика строится от таза и поясницы: таз смещается по горизонтали, вращается и опускается на переносе веса; позвоночник мягко компенсирует движение; бёдра работают независимо; ноги ставятся по узкой перекрёстной линии. Колени, стопы, носки и каблуки имеют отдельные фазы, поэтому это не обычная «мужская» походка с добавленным покачиванием корпуса.

## Готовые клипы

| Клип | Длина | Назначение |
| --- | ---: | --- |
| `Idle_Elegant` | 2,40 с | Спокойная стойка с дыханием и лёгким переносом веса; цикл. |
| `Walk_Catwalk_InPlace` | 1,52 с | Модельная походка на месте; бесшовный цикл. |
| `Walk_Catwalk_Forward_Root` | 1,52 с | Два шага вперёд с root motion примерно 1,55 м. |
| `Walk_Backward_Root` | 1,52 с | Контролируемое движение назад лицом вперёд. |
| `Walk_Away_Root` | 1,52 с | Модель уже развёрнута спиной и уходит. |
| `Turn_180_Left` | 1,36 с | Разворот на 180° влево с работой стоп и таза. |
| `Turn_180_Right` | 1,36 с | Зеркальный разворот вправо. |
| `Look_Back_Over_Shoulder` | 3,20 с | Оборачивается через плечо и возвращает взгляд. |
| `Approach_Turn_Look_Leave` | 9,00 с | Полная сцена: подходит, останавливается, разворачивается, оборачивается и уходит. |

Частота ключей — 30 FPS. Оси: `+Y` вверх, лицо и движение вперёд — по `+Z`. Высота модели — 1,90 м.

## Подключение в Godot

1. Ресурсы находятся в `res://assets/lost_signal/characters/woman_catwalk/`.
2. PBR-модель подключена через `woman_catwalk_pbr.tscn` к официантке в сцене закусочной.
3. На обёртке уже стоит `woman_animation_controller.gd`; по умолчанию включается `Idle_Elegant`.
4. Для полной постановочной сцены вызовите:

```gdscript
$WomanCatwalkPBR.play_full_sequence()
```

Другие короткие вызовы: `play_walk_in_place()`, `play_forward()`, `play_backward()`, `turn_left()`, `turn_right()`, `look_back()` и `play_away()`.

## Важное про перемещение

`Walk_Catwalk_Forward_Root`, `Walk_Backward_Root`, `Walk_Away_Root` и полный девятисекундный клип содержат root motion прямо в скелете. Не перемещайте `CharacterBody3D` вторым независимым кодом во время этих клипов — иначе получится двойное движение. Для NPC с навигацией используйте `Walk_Catwalk_InPlace`, а скорость задавайте родительскому `CharacterBody3D` из кода.

Если модель нужна как фоновой NPC, PBR-версию разумно оставить возле камеры, а shaded-версию — на средней и дальней дистанции.
