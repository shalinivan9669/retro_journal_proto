# Diner Poly Haven Assets

All assets in this folder are from Poly Haven and are released under CC0.

| Asset | Use | Source |
|---|---|---|
| `outdoor_table_chair_set_01` | Five interior table/chair sets | https://polyhaven.com/a/outdoor_table_chair_set_01 |
| `drawer_cabinet` | Cabinet at the rear counter wall | https://polyhaven.com/a/drawer_cabinet |
| `wooden_display_shelves_01` | Display shelving at the rear wall | https://polyhaven.com/a/wooden_display_shelves_01 |
| `modern_arm_chair_01` | Chair outside the entrance | https://polyhaven.com/a/modern_arm_chair_01 |
| `wood_cabinet_worn_long` | Diner floor PBR material | https://polyhaven.com/a/wood_cabinet_worn_long |

Runtime import uses the 2K glTF/JPG variants. The floor uses diffuse, normal (GL), roughness and AO. Displacement is intentionally omitted for performance.
