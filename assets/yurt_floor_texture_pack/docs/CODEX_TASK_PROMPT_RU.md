Ты работаешь с архивом yurt_floor_texture_pack. Нужно собрать красивый, богатый, многослойный пол юрты для проекта на Godot 4.

Цели:
1. Пол должен выглядеть как layered floor в юрте: мягкий войлочный настил + главный декоративный ковер + бордюры + отдельные сидячие коврики у стен.
2. Использовать все 4 набора материалов, но не в равной степени. Главные наборы — B_ornamental_pattern и C_felt_pile. A_blue_fabric и D_checkered использовать как вторичные акценты.
3. Если синий материал выглядит слишком как джинса, использовать подготовленные recolor-варианты из `A_blue_fabric/derived`.
4. Прочитать `docs/YURT_FLOOR_COMPOSITION.md` и следовать общей композиции.
5. Собрать пол из нескольких отдельных mesh-плоскостей с малым смещением по Y, а не одной текстуры на всю юрту.
6. Проверить реальные роли EXR-карт и при необходимости переназначить их как normal / roughness / height / displacement / mask.
7. Если какие-то карты неочевидны, не ломать сцену: использовать только подтвержденные карты, а остальные оставить как дополнительные.
8. Сохранить итоговую сцену чистой и понятной: именованные mesh-узлы, материалы, папки.

Ожидаемый результат:
- В Godot сцена с красивым полом юрты.
- Материалы разложены по отдельным коврам.
- Есть главный центральный ковер, центральный медальон, бордюрное кольцо и периферийные коврики.
- Вид богатый, теплый, красно-молочно-серый, с очень небольшими вторичными акцентами.
