# Yurt Floor Texture Pack

Этот архив собран так, чтобы его можно было отдать Codex для дальнейшей сборки пола юрты в Godot.

Что внутри:
- `textures/A_blue_fabric` — синий тканевый набор. Лучше использовать как небольшой акцент или в перекрашенном виде.
- `textures/B_ornamental_pattern` — орнаментальный набор. Лучший кандидат на главный декоративный ковер или бордюр.
- `textures/C_felt_pile` — войлок / ворс / шерсть. Лучший кандидат на базовый настил и центральный красный ковер.
- `textures/D_checkered` — красно-белая клетка. Лучше как локальные коврики, сидушки или вторичный акцент.
- `docs/` — манифест, план композиции, подсказки для Godot и готовый промт для Codex.

Важно:
- Имена `basecolor`, `roughness`, `height`, `detail`, `pile` — это **оценка по визуальному виду**, а не 100% подтверждение.
- Все EXR-файлы сохранены как есть. Пусть Codex / Blender / Godot сам проверит их реальную роль в шейдере.
- Для суперреализма главный упор надо делать на `C_felt_pile` и `B_ornamental_pattern`.
- `A_blue_fabric` в чистом синем виде выглядит менее аутентично для юрты, поэтому я добавил перекрашенные варианты в `derived/`.
