Clean Yurt Replacement

1. Disabled old yurt nodes
- The old runtime-generated yurt is no longer built in Main.tscn.
- The previous Room node was renamed to:
  OldBrokenYurt_Disabled
- Its yurt_room_builder.gd script was removed from the Main.tscn instance, so it no longer creates old walls, roof panels, shanyrak beams, the old sky disc, or old wall/floor collisions at runtime.

2. New yurt model
- Model path:
  res://assets/models/yurt/clean_lowpoly_yurt_with_open_shanyrak.glb
- Main.tscn instance:
  CleanYurt
- Position:
  Vector3(0, 0, 0)
- Scale:
  Vector3(1, 1, 1)
- Rotation:
  180 degrees around Y, so the model entrance faces the existing door/exit at negative Z.

3. Materials
- Walls and floor:
  res://materials/mat_yurt_wall.tres
- Roof:
  res://materials/mat_yurt_roof.tres
- Shanyrak, roof beams, and wood belt:
  res://materials/mat_yurt_wood_dark.tres
- Material overrides are set in Main.tscn on CleanYurt/world child mesh nodes.

4. Manual collisions
- Collision root in Main.tscn:
  YurtCollision
- Created simple BoxShape3D collisions:
  YurtFloorCollision
  YurtWallCollision_01
  YurtWallCollision_02
  YurtWallCollision_03
  YurtWallCollision_04
  YurtWallCollision_05
  YurtWallCollision_06
  YurtWallCollision_07
  YurtEntrance_LeftCollision
  YurtEntrance_RightCollision
  YurtEntrance_TopCollision
- No automatic mesh collision is used from the GLB.
- The roof and shanyrak do not have collision, so the camera and player do not catch on roof beams.

5. Where to change yurt size
- Change the CleanYurt transform in:
  res://scenes/Main.tscn
- Prefer keeping scale at Vector3(1, 1, 1).
- If scale changes, update every BoxShape3D under YurtCollision manually.

6. Where to change shanyrak opening size
- The shanyrak opening is part of the GLB mesh:
  res://assets/models/yurt/clean_lowpoly_yurt_with_open_shanyrak.glb
- To change its size, edit or replace the GLB model.
- There is no collision over the shanyrak.

7. Where to change roof material
- Open:
  res://materials/mat_yurt_roof.tres
- Or change the material overrides for YurtRoofSegment_00 through YurtRoofSegment_07 in:
  res://scenes/Main.tscn

8. If the sky is not visible through the shanyrak
- Check that OldBrokenYurt_Disabled has no active yurt_room_builder.gd script.
- Check that CleanYurt has no central cap mesh covering the opening.
- Check that SkyDome is still generated by SteppeEnvironment.
- Check res://materials/mat_sky_dome.tres cull_mode and shading settings.
- Check that no new collision or MeshInstance3D was placed over the top opening.

9. If the player gets stuck in the entrance
- Check YurtEntrance_LeftCollision and YurtEntrance_RightCollision positions.
- Keep the gap between those two side collisions wider than the player capsule.
- Check YurtEntrance_TopCollision height; it should stay above the player head.
- Check that the old Room/yurt generated collisions are not active.
- Check that InactiveDoor is still an interactable portal object and not the only physical way out.
