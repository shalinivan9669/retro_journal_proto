If a texture still looks stretched:
- First adjust uv1_scale in the relevant res://materials/*.tres file.
- For walls and floor, increase X/Y scale to repeat the texture more.
- For door, cube, and WallOppositePanel, keep scale near Vector3(1, 1, 1) unless you intentionally want tiling.
- If material UV scale is not enough, create fitted textures under res://assets/textures/fitted/ from the already imported project images.
