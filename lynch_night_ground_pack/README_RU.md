# Ночная земля для сцены со вспышками — Godot 4.7

Это не «коричневая игровая грязь», а почти бесцветная каменистая земля, которая остаётся читаемой во время коротких вспышек. Пакет не меняет небо, трассеры, облака или взрывы.

## Что внутри

- `hero_rocks_ground_08` — главная 4K PBR-поверхность: мелкий щебень, плотная земля, редкие влажные углубления.
- `dry_mud_field_001` — 2K спокойная вариация для разрыва повторения основной текстуры.
- `muddy_tracks` — 2K редкие локальные участки развороченной влажной земли. Не покрывать ими всю сцену.
- `processed/hero_night_albedo_4k.jpg` — подготовленный холодный, почти нейтральный albedo. Он затемнён умеренно: детали не исчезнут под вспышками.
- `processed/hero_wet_pockets_mask_4k.png` — маска влажных углублений.
- `godot/lynch_night_ground.gdshader` — смешивание трёх поверхностей в мировых координатах, макровариация и локальная влажность.
- `godot/lynch_night_ground_material.tres` — уже собранный материал.

## Установка

1. Скопировать папку `lynch_night_ground_pack` целиком в корень Godot-проекта: `res://lynch_night_ground_pack/`.
2. Дождаться импорта текстур.
3. Назначить `godot/lynch_night_ground_material.tres` в `material_override` основного `MeshInstance3D` земли.
4. Карты `*_nor_gl_*` должны импортироваться как Normal Map. `rough`, `ao`, `disp` и `mask` — без sRGB.
5. Для крупного ландшафта геометрию холмов и разрывов оставить в самом mesh/terrain. `vertex_displacement_m` по умолчанию равен `0`: 4K height-карта не должна создавать большие холмы.

## Стартовые значения для нужного кадра

- `dry_amount = 0.34`
- `mud_amount = 0.16`
- `wet_amount = 0.52`
- `wet_roughness = 0.24`
- `normal_strength = 0.78`
- `macro_contrast = 0.13`

Нормальный диапазон: основная земля 70–80%, сухая вариация 15–25%, мокрая развороченная грязь 5–10% видимой площади. Если колеи видны постоянно, результат сразу начинает выглядеть как повторяющаяся игровая текстура.

## Свет

Землю не надо давить в абсолютный чёрный через albedo. Ночь создаётся низким окружением, а поверхность на доли секунды раскрывается уже существующими вспышками. Для WorldEnvironment разумная база: ACES, фон очень низкий, без чрезмерного glow. Во время вспышки roughness влажных карманов даст короткие холодные блики; именно это создаёт ощущение сна, а не мультяшный контраст.

Если сцена стала слишком серой, сначала уменьшить энергию окружения, а не ещё сильнее затемнять albedo. Если вспышки выглядят плоско — поднять `normal_strength` до `0.9`, а не добавлять резкость в картинку.

## Производительность

Материал делает несколько PBR-выборок и рассчитан на геройскую землю рядом с камерой. Для дальних участков используйте только `hero_night_albedo_4k` + normal + roughness в обычном StandardMaterial3D или сделайте упрощённую копию материала без dry/mud слоёв.
